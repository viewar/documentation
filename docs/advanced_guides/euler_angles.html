Euler angles

In mathematics, the Euler angles are three angles introduced by Leonhard Euler to describe the orientation of a rigid body with respect to a fixed coordinate system.

Any orientation can be achieved by composing three elemental rotations, i.e., rotations about the axes of a coordinate system. Euler angles can be defined by three of these rotations, typically denoted as: { α, β, γ } { φ, θ, ψ } { yaw, pitch, roll }
